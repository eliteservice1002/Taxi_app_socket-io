# WeCode - Socket.io Backend
A React-Native App that links Taxi Drivers and Passengers. 

Socket.io server to connect Passengers and Drivers in real time.

## Installation

From terminal install dependencies

```
yarn
```

## Running

```
$ node index.js
```
